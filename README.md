# Portfólio Profissional – Cícero Quintino Junior

Este repositório contém meu portfólio pessoal feito em Jekyll, publicado via GitHub Pages e pronto para domínio próprio.

## 📚 Sobre

Aqui você encontra minhas experiências, projetos, habilidades e formas de contato, tudo apresentado de forma moderna e responsiva.

## ✨ Recursos

- Seções dinâmicas (Sobre, Projetos, Skills, Contato)
- Design responsivo e dark mode
- SEO avançado e integração com Google Analytics
- Pronto para domínio customizado

## 🚀 Como editar

1. **Clone este repositório**
2. Edite os arquivos em `_sections/`, `_config.yml` e as imagens em `assets/images/`
3. Faça push para o GitHub e publique pelo GitHub Pages!
